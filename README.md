👁️ BlinkWise

AI-Powered Personal Eye Health Assistant

BlinkWise is a desktop application that uses your webcam to monitor your eye blink rate in real time. It helps reduce digital eye strain by tracking blinking behavior and providing smart reminders to encourage healthier screen habits.

About
People often blink much less while coding, studying, gaming, or working on a computer. Reduced blinking can lead to:
- 👁️ Dry eyes
- 😣 Eye strain
- 🔴 Eye irritation
- 👓 Long-term vision discomfort
BlinkWise aims to solve this problem by acting as a **Personal Eye Health Assistant**, using computer vision to monitor blink activity and provide health insights.

⚙️ Installation
Clone the repository:
```bash
git clone https://github.com/donthulaakshara/BlinkWise1.git
```
Move into the project folder:
```bash
cd BlinkWise1
```
Install dependencies:
```bash
pip install -r requirements.txt
```
Run the application:
```bash
python main.py
```
